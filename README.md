# Maseno University Facility Access Map

An interactive web map displaying facilities and features at Maseno University, created with QGIS2web and Leaflet.

## Features

- Interactive map visualization of campus facilities
- Layers including:
  - Buildings
  - Roads
  - Trees
  - Grass areas
  - Parking
  - Farm areas
  - Flowers
  - Fields
  - Bare ground
  - Amenities

## Technologies

- [Leaflet](https://leafletjs.com/) - Interactive mapping
- [QGIS2web](https://github.com/qgis2web/qgis2web) - Map generation tool
- HTML5, CSS, JavaScript

## Usage

Simply open `index.html` in a modern web browser to view the map.
